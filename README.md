北海 · 网易云风格音乐播放器

一个现代化的仿网易云风格音乐播放器，具有精美的UI设计和流畅的交互体验，支持本地和在线音乐播放、歌词同步显示、播放列表管理等功能。

🎯 功能特性

🎵 核心功能

· 黑胶唱片播放器：拟真黑胶唱片播放动画，支持旋转和滑动手势切换
· 歌词同步显示：自动解析LRC歌词文件，支持点击跳转和高亮同步
· 多播放模式：顺序播放、单曲循环、随机播放三种模式
· 播放历史：自动记录播放历史，支持历史回放
· 高清音质：支持极高音质音频播放

🎨 视觉设计

· 网易云风格UI：简洁美观的界面设计，精心调校的动画效果
· 响应式布局：完美适配移动端和桌面端
· 夜间歌词模式：全屏歌词显示，专注音乐体验
· 黑胶唱片动画：拟真的唱片旋转和滑动切换效果

🔧 实用功能

· 歌曲分享：支持Web Share API和复制链接分享
· 播放列表：完整播放列表管理，支持历史记录查看
· 进度控制：点击进度条跳转，高潮时间点标记
· 滑动操作：支持左右滑动手势切换歌曲
· 离线缓存：智能缓存歌词和歌曲数据

🚀 快速开始

环境要求

· 服务器：支持PHP的Web服务器（Apache/Nginx）
· PHP版本：PHP 7.4或更高版本
· 浏览器：支持ES6的现代浏览器
· 文件权限：songs.json文件需要读写权限

安装步骤

1. 克隆或下载项目
   ```bash
   git clone <repository-url>
   ```
2. 配置服务器
   · 将项目文件放置在Web服务器目录中
   · 确保songs.json文件可写（权限设为664或666）
   · 如果使用Nginx，确保支持.lrc文件的MIME类型
3. 初始化歌曲数据
   · 编辑songs.json文件，按照示例格式添加歌曲
   · 或直接访问song_manager.php进行可视化添加
4. 准备资源文件
   · 将SVG图标放置在对应目录
   · 准备歌词文件（可选）
   · 确保音频文件的URL可正常访问
5. 访问应用
   · 播放器：http://your-domain.com/index.html
   · 管理后台：http://your-domain.com/song_manager.php

📱 使用说明

🎵 基本播放控制

1. 播放/暂停：点击中间的播放按钮
2. 上一首/下一首：点击两侧箭头按钮或左右滑动唱片
3. 切换播放模式：点击左上角模式按钮
4. 进度控制：点击进度条任意位置跳转

📖 歌词功能

1. 显示歌词：点击唱片区域切换歌词模式
2. 歌词跳转：在歌词模式下，点击任意歌词跳转到对应时间
3. 返回唱片：点击歌词区域任意空白处返回

📋 播放列表

1. 打开列表：点击右下角列表按钮
2. 切换歌曲：在列表中点击任意歌曲立即播放
3. 查看历史：切换到"播放历史"标签页
4. 关闭列表：点击空白区域或关闭按钮

🛠️ 歌曲管理

1. 添加歌曲：访问song_manager.php，填写表单后提交
2. 编辑歌曲：在管理页面点击"编辑"按钮修改信息
3. 删除歌曲：点击"删除"按钮移除歌曲
4. 批量管理：支持批量导入导出（高级功能）

⚙️ 数据格式说明

songs.json结构

```json
[
  {
    "id": 1,
    "title": "歌曲标题",
    "artist": "歌手名称",
    "cover": "封面图片URL",
    "audio": "音频文件URL",
    "lrc": "歌词文件路径",
    "climax": "高潮时间点（格式：MM:SS）",
    "duration": "歌曲时长（格式：MM:SS）",
    "quality": "音质描述"
  }
]
```

LRC歌词格式

```lrc
[00:00.00]歌曲标题 - 歌手
[00:10.00]第一句歌词
[00:15.00]第二句歌词
[01:25.00]高潮部分歌词
```

🐛 故障排除

常见问题

1. 音频无法播放
   · 检查音频URL是否可访问
   · 确认浏览器支持音频格式
   · 查看控制台是否有CORS错误
2. 歌词不显示
   · 检查歌词文件路径是否正确
   · 确认LRC文件格式规范
   · 查看网络请求状态
3. 歌曲管理页面无法保存
   · 检查songs.json文件权限
   · 确认PHP版本支持JSON操作
   · 检查表单数据完整性
4. 移动端手势无效
   · 确认触屏事件支持
   · 检查页面是否被缩放
   · 更新浏览器到最新版本

调试工具

· 按F12打开开发者工具
· 查看控制台输出
· 检查网络请求状态
· 使用手机调试模式
